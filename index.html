About Me

My name is Chad Scarborough. I'm a former academic looking to transition to software engineering. In my roughly decade of programming as a hobby, I've been involved in a wide variety of projects, from game development to avant-garde art installations. It's important to me that the work I do has the broadest possible positive impact on the world, and I believe software development will allow me to achieve that ideal.

Projects

The BEAM Piano Project

An art installation where one controls a light show by playing a prepared piano. The piano keys simulate button presses on a number of game controllers, all of which are connected to a Raspberry Pi. The inputs from these controllers are processed by a set of Python scripts, causing the Raspberry Pi to output various lighting effects.

Monopoly API

A Javascript API for generating statistical data over a large number of simulated games of Monopoly with a specified number of games, turns per game, and opponents.
